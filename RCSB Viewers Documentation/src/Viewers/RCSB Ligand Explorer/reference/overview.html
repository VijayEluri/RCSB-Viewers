The Ligand Explorer presents a structure and non-structure molecules, with a user interface panel to highlight and display several measurements between atoms of a ligand and atoms of the structure.

This viewer implements a number of extended classes to provide things like annotated atoms and interactions. These features might be useful to be pushed down into the Viewer Framework, or lower.

Conversely, there has been discussion of creating a single 'unified' viewer. This viewer would probably be the best starting point for such an effort - it shouldn't be too difficult to pull the Protein Workshop control panel in, and have another selection in the menu item to hide controls, entirely - providing Simple Viewer functionality.

Rendering styles would have to be changed when entering and exiting Ligand Explorer mode.
