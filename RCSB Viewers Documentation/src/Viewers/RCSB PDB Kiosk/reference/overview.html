The PDB Kiosk is a 'show' viewer - it runs through a set of structures and displays them in slide-show fashion, moving the camera in smooth animations to view the current structure from different angles and attributes. In addition, it will focus in on any non-structural molecules. A panel at the base provides author and other information.

This viewer is driven by either:

  • a file in it's current directory (called screensaver.properties) to point to a directory containing structure files.
  • a file in that directory called files.txt, naming files or bare structure ids
  • a list of structures provided after the argument -structure-id-list, comma separated.

When only structure id's are present, they are downloaded from the RCSB PDB Website. See the source documentation for details.

Note that to debug the PDB Kiosk, you have to use two machines and debug remotely, since it puts the display in full-screen mode. Read the Eclipse documentation (or whatever development environment you're working in) for instructions on how to do that.
