The RCSB Viewer Framework provides UI and OpenGL components to create 3d applications.

It depends on and cooperates with the RCSB UIApp Framework.

Source is broken up into the following categories:

Controllers
Extensions of {@link org.rcsb.uiapp} and new controllers, specifically tuned to 3d viewer appliations. Overrides of UIApp classes are prefixed with VF. Any viewers should override the VF- versions of classes, if available.
GL Scene
OpenGL specific packages.
UI
Extensions of {@link org.rcsb.uiapp} UI modules, prefixed with VF.
