The UIApp Framework provides components necessary to make a UI (windowing) application. Note that it is not specifically tied to any particular visualization technology. It is dependent on Swing.

In particular, there are no 2d or 3d graphics modules, here. The RCSB Viewer Framework provides additional functionality to create 3d viewer applications.

Source directories in the project break out as follows:

Controllers ({@link org.rcsb.uiApp.controllers})
Controller modules to control assembly, sequencing, module behavior and inter-relationships, etc. The application class (super-controller) is defined, here.
UI ({@link org.rcsb.uiApp.ui})
UI components including the mainframe, document frame, dialogs, progress panel, etc.
Structure Model
Extensions of the {@org.rcsb.mbt.model} classes, prefixed with UIApp. Applications should derive from these classes, rather than the MBT base classes.
