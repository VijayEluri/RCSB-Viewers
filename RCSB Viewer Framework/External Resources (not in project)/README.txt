This mainly contains stuff used to generate images, etc, but not included in the project.

icon_128_viewers.psd -
  A composite photoshop file of the viewer's icons.  Consists of a base layer and
  viewer specific image layers.  Turn on the layers you want for the icon (add new
  ones as necessary) and save in the appropriate project 'images' directory as a
  .png file
