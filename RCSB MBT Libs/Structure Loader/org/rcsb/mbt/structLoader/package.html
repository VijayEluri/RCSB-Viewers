File loaders are contained in this package. Currently, only two filetypes are supported:

  • .pdb - the original 80 character punchcard image format.
  • .xml - an xml representation of .cif data.

Note the {@linkplain org.rcsb.mbt.structLoader.IStructureLoader} interface. This primarily defines common accessors to resultant information from the loading operation.

Both the {@linkplain org.rcsb.mbt.structLoader.PdbStructureLoader} and the {@linkplain org.rcsb.mbt.structLoader.StructureXMLHandler} are capable of loading gzip'ed files.
